**Advanced Calculator** is a advanced calculator to help you with your IXL Lessons and Math Homework with many different modes to choose from

### Download

You can download the calculator by going to the main page of the github and clicking on the latest release and downloading the file called **Advanced-Caculator.exe** after that run it. You must have Java 8 installed on your computer for the application to work!

### IXL Lessons

I have made this calculator to help you with your IXL lesson we will be inputting a IXL Mode where you will input Grade, State, How many months have school have you been in, it will then supply you over basic standards that you should be learning you will be able to select them and you will find modes that will help you out on your lessons
